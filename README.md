# Powered-Applications
강력한 머신러닝 애플리케이션 구축 과정 배우기

## Goal
- 사용자의 질문을 받고 더 나은 문장으로 개선하는 에디터 개발
- 질문하는 방식이 유용한 답을 얻는 데 큰 영향을 미침
- 더 나은 답을 찾기 위해서는 "더 나은 질문을 하도록 돕는 에디터를 만드는 것"
